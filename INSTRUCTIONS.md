## README.md Requirements
1. Name of the project
2. Names of team members
3. Installation/compilation instructions
4. Description of the API and potential endpoints
5. Example usage of the API or interface
6. Disclosure of the use of AI and links to any chat logs (if none was used, please say so)
7. Disclosure of the use of tutorials, starter code, or other external resources that were used to help write the code for the API

*Any submissions missing any of these requirements are disqualified from prizes. Any projects caught plagiarizing code will also be disqualified.*

## Rules
1. The entire API must be made in the span of 1 hour and submitted to this repository along with the README.md
2. AI usage is allowed. Vibe coding is not allowed
3. A majority of the API's functionality code must be written in C++. User Interface is exempt
4. Team sizes of only 1-4 people are allowed
5. Only the latest commit made at the time of the deadline (1pm Thursday) will be evaluated. Any commits made after this time cutoff are not evaluated.
6. Code from previously submitted assignments (e.g. labs or modules) is also allowed
